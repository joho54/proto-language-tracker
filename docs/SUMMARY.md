## 1. 프로젝트 정체 (재정의됨)

**원래 목표**: PIE에서 음변화를 배워 한국어 등 조어를 재구하고, 독립 재구된 조어들의 형태론적
동형성이 우연을 초과하는지 검정 → 한-타밀 등 심층 관계 *증명*.

**POC가 강제한 재정의**: 심층 관계의 *증명*은 정보이론적으로 막혀 있다(§4). 대신 방어 가능하고
신규한 정체는 —

> **"관계를 증명하는 기계"가 아니라, 언어 변화를 *규칙 + 유형화된 불규칙*으로 분해하고
> 그 위에서 *판정가능성 지평*을 재는 계측기.**

> ★ **최종 갱신(§4.1)**: 더 파보니 이마저도 좁다. 계산은 엔트로피(파괴된 정보)를 못 되돌리고
> 증인만이 통화다 → 진짜 자리 = **삼각측량(고고학·유전학)의 *언어학 다리* = 시나리오에 *정직한
> 신뢰구간*을 다는 계측기.** Robbeets가 Nature 받고 반박당한 그 빠진 오차범위가 우리 자산.

## 2. 중심 명제 (신규 방법론) ★

언어 변화를 두 부분으로 분해한다:

```
관측 = 규칙적 음변화(신문법 baseline)  +  유형화된 일탈(deviation)
                                          ├ 차용(borrowing)   → 접촉 신호
                                          ├ 유추(analogy)     → 형태론 압력
                                          ├ 산발변화(metathesis…) → 국소 과정
                                          └ 혁신/노이즈        → 잔차 = 지평
```

핵심 주장 3가지:
1. **"불규칙"은 절대값이 아니라 *적합된 규칙 모델에 대한 잔차*** 로 정의된다.
2. 각 일탈을 *유형화*하면 **(a) 계보 신호가 깨끗해지고 (b) 그 유형 자체가 새 정보 채널**이 된다.
3. 모든 유형으로 귀속하고 **남은 잔차 = 진짜 노이즈 = 판정 지평**. null 모델이 이를 정량화하며,
   표현(이산/연속)·방법과 무관하게 falsifiability를 떠받친다.

**신규성**(deep-research 확인): 차용탐지·borrowing-aware 계통학·자동 동원어탐지는 *각각* 존재하나,
이들을 **하나의 유형-혼합 분해 + 계보 정화 + 유형별 신호복원 + null 지평**으로 묶은 선행연구는 없음.

### 2.1 유형을 어떻게 두는가 — 손코딩 ❌, 발견·판정 ✓ (POC-14 검증)

일탈 유형을 일일이 손으로 추가하지 않는다. **일반 기제 2개 + 노이즈**:
- **M1 끌림(attraction-to-source)**: 우도 `P_adapt(w|s)` 하나 + **출처 카탈로그 A_i** 확장.
  차용=donor 카탈로그 / 유추=패러다임 cell / 혼성=어휘 이웃 — *코드가 아니라 카탈로그 태그*로 흡수.
- **M2 국소연산**: 음위전환·동화 등의 *닫힌* 작은 라이브러리(보조데이터 불요).
- **잔차 = 노이즈 = 지평**(긴 꼬리는 모델링 안 함).
- 채널 둘: `R(·)` 깊은 규칙변화 / `Adapt(·)` 얕은 적응.
- **에피사이클 차단 = MDL/절약 게이트**: 유형은 기술길이를 우연 이상 줄여야 채택. 닭-달걀은 **EM 부트스트랩**.
→ POC-14: 비지도 EM이 차용층 발견(gold 76%) + MDL이 2-모드 채택 = **유형은 데이터가 발견·판정.**
→ POC-15 갱신: M1 끌림은 **귀속(어느 출처)엔 생성 우도, 탐지(차용인가)엔 판별통계**가 필요 —
  카탈로그-조회 우도는 phonotactic을 항상 이겨 탐지서 붕괴(아래 §3, PIPELINE.md POC-15). 둘을 분리.

## 4. 깊이의 벽 — negative-results 지도 (그 자체로 기여)

심층 *증명*이 왜 막히는지를 **모호한 통념에서 측정값으로** 전환:
- **정보 한계**(POC-10): 표현(이산=연속)·블랙박스로 못 넘음. 파괴된 정보는 복원 불가.
- **동원어 개수 한계**(POC-6/11): 지평은 비율 아닌 *개수*(~15-20쌍); 안정어 타겟팅은 wash.
- **단계적 재구 실패**(POC-5c): 점추정 연쇄가 오차 누적으로 상쇄.
- **형태론 비구원**(POC-9, B-scout/Georg): 형태소는 어휘보다 *더* 우연충돌 가능; 패러다임 붕괴(Robbeets).
- **두 정보손실 기제**: 어휘교체(공통조상 단어 소멸) + 음운병합(구별 소멸).
- **유형론 baseline 교란**(POC-17b, 신규): 순열-쌍셔플 null은 *쌍짓기*만 통제, **유형론/areal 유사성은 미통제**.
  실데이터서 무관 Uralic이 Japonic과 동급 신호 → 계보 검정엔 *유형론-매칭 대조군*이 필수(쌍셔플 null만으론 부족).

### 4.1 정보이론적 바닥 — 엔트로피와 도구의 재배치 (2026-06-16, 개념 정점)

위 한계들을 *하나의 원리*로 통합한 결론. 대화로 도달, 차후 실험으로 정식화 대상.

**① 벽은 질문-독립적이다.** 계보든 접촉이든 *같은 "재구가능 깊이" 봉투*에 갇힌다. proto보다 깊은(=재구 너머)
신호는 계보라 부르든 접촉이라 부르든 *같은 정보한계*로 안 보인다. 양성(접촉) 전환은 새 깊이를 사주지 않고
(POC-18: ASJP 접촉신호 약함·유형론 미분리), 같은 봉투 안에서 *질문*만 바꾼다. 신규성이 사는 띠 = 표면(이미
연구됨)과 재구 너머(불가능) 사이 ≈ **proto-to-proto 접촉 ~2–4ky**(Transeurasian 논쟁이 사는 좁은 띠).

**② falsifier ↔ 재구기 = 하나의 대응모델을 반대로 읽은 쌍대.** falsifier의 심장 = 분절 대응표 `P(x,y)`,
MI는 그걸 스칼라로 collapse한 것. collapse 안 하고 들면 재구기(POC-5/6 같은 `align` 코드 공유). **그러나
`MI(스칼라) ⟺ 대응표(구조) ⟺ 재구가능성`은 같은 정보량의 세 얼굴** — 신호 floor면 대응표는 노이즈→
재구는 *허구* 산출(POC-5b 확인). 방향을 틀어도 깊이는 0mm 안 늘고, 트는 게 사는 건 *교정*(어디서 믿고
어디서 기권할지=calibrated abstention, POC-4 엔트로피-기권 성숙판)이지 깊이가 아니다.

**③ 엔트로피는 "극복"이 아니라 "수지균형".** 병합(`*p,*b→p`)으로 파괴된 비트는 단일 계열서 비가역(Maxwell의
악마); 표현·prior로 *1비트도 못 만든다*. 유일한 통화 = **병합이 겹치지 않은 독립 증인**이 바깥에서 비트 공급
(POC-4: 자매 3개→entropy 0). **깊이 = 증인 공급 − 누적 엔트로피.** 벽 = 증인이 엔트로피 밑으로 떨어지는 곳.
POC-17e가 증명: IE(증인 多)→6ky 재구 MI 1.2; 한-일(Koreanic 자매 枯渴)→floor. **방법이 약한 게 아니라
증인이 소진됨.** 가장 안 쓰인 증인 채널 = **차용 화석**(빌린 단어가 공여 proto의 옛 상태를 박제).

**④ 증인은 땅에서 — 그러나 한-일엔 닫혀 있다.** 새 1차 증인은 고고학에서: ①금석문·해독(저-엔트로피 증인;
히타이트→PIE 후두음 확정의 결정적 선례) — **단 문자 必**; ②물질고고학·aDNA — *형태*는 못 주고 *시나리오*
(접촉/이주 有無·시점)만. 한-일의 결정 시기(~3–5ky)는 **무문자**라 ① 닫힘, ②만 남음.

**⑤ ∴ 도구의 재배치(프로젝트 정체성 갱신).** 계산은 증인을 *만들* 수 없다. 진짜 자리 = **삼각측량의 언어학
다리** — (a) 벽이 *어디*인지 정밀 측정(판정지평), (b) *기존* 증인(특히 차용 화석)서 신호 최대 추출, (c)
고고학·유전학이 시나리오를 대면 *언어 신호가 정합한지 정직한 오차범위로 검정*. ★Robbeets가 Nature 받고도
Tian/Vovin에 반박당한 핵심 = (c) 부재(오차범위 없는 단정). **우리가 만든 것(유형론통제·null·검정력 사다리·
지평)이 그 빠진 신뢰구간.** → 프로젝트 = "관계 증명기"도 "falsifier"도 아닌, **삼각측량에 신뢰구간을 다는
계측기.** Nature급은 계산 단독 아닌 *삼각측량*에 있고, 우리 고유 기여 = 거기 *처음으로 정직한 오차범위*를 다는 것.

```
사슬: Nature 꿈 → 계산 벽(엔트로피) → 표현·prior 무효(POC-10/5b) → 양성전환도 같은 벽(POC-18)
 → falsifier↔재구기 쌍대=같은 MI(POC-5b/6/17e) → 엔트로피=수지균형, 증인이 유일통화(POC-4)
 → 증인은 땅에서(①문자必→한-일 닫힘 ②aDNA=시나리오만) → ∴ 도구=삼각측량의 언어학 다리+정직한 오차범위
```

→ 한-타밀/일본(~5천년+)은 보존율이 지평 밑 → 모델이 "판정불가+정량한계"를 반환하는 게 정직한 답.
