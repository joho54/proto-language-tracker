## 1. 프로젝트 정체 (재정의됨)

**원래 목표**: PIE에서 음변화를 배워 한국어 등 조어를 재구하고, 독립 재구된 조어들의 형태론적
동형성이 우연을 초과하는지 검정 → 한-타밀 등 심층 관계 *증명*.

**POC가 강제한 재정의**: 심층 관계의 *증명*은 정보이론적으로 막혀 있다(§4). 대신 방어 가능하고
신규한 정체는 —

> **"관계를 증명하는 기계"가 아니라, 언어 변화를 *규칙 + 유형화된 불규칙*으로 분해하고
> 그 위에서 *판정가능성 지평*을 재는 계측기.**

---

## 2. 중심 명제 (신규 방법론) ★

언어 변화를 두 부분으로 분해한다:

```
관측 = 규칙적 음변화(신문법 baseline)  +  유형화된 일탈(deviation)
                                          ├ 차용(borrowing)   → 접촉 신호
                                          ├ 유추(analogy)     → 형태론 압력
                                          ├ 산발변화(metathesis…) → 국소 과정
                                          └ 혁신/노이즈        → 잔차 = 지평
```

핵심 주장 3가지:
1. **"불규칙"은 절대값이 아니라 *적합된 규칙 모델에 대한 잔차*** 로 정의된다.
2. 각 일탈을 *유형화*하면 **(a) 계보 신호가 깨끗해지고 (b) 그 유형 자체가 새 정보 채널**이 된다.
3. 모든 유형으로 귀속하고 **남은 잔차 = 진짜 노이즈 = 판정 지평**. null 모델이 이를 정량화하며,
   표현(이산/연속)·방법과 무관하게 falsifiability를 떠받친다.

**신규성**(deep-research 확인): 차용탐지·borrowing-aware 계통학·자동 동원어탐지는 *각각* 존재하나,
이들을 **하나의 유형-혼합 분해 + 계보 정화 + 유형별 신호복원 + null 지평**으로 묶은 선행연구는 없음.

### 2.1 유형을 어떻게 두는가 — 손코딩 ❌, 발견·판정 ✓ (POC-14 검증)

일탈 유형을 일일이 손으로 추가하지 않는다. **일반 기제 2개 + 노이즈**:
- **M1 끌림(attraction-to-source)**: 우도 `P_adapt(w|s)` 하나 + **출처 카탈로그 A_i** 확장.
  차용=donor 카탈로그 / 유추=패러다임 cell / 혼성=어휘 이웃 — *코드가 아니라 카탈로그 태그*로 흡수.
- **M2 국소연산**: 음위전환·동화 등의 *닫힌* 작은 라이브러리(보조데이터 불요).
- **잔차 = 노이즈 = 지평**(긴 꼬리는 모델링 안 함).
- 채널 둘: `R(·)` 깊은 규칙변화 / `Adapt(·)` 얕은 적응.
- **에피사이클 차단 = MDL/절약 게이트**: 유형은 기술길이를 우연 이상 줄여야 채택. 닭-달걀은 **EM 부트스트랩**.
→ POC-14: 비지도 EM이 차용층 발견(gold 76%) + MDL이 2-모드 채택 = **유형은 데이터가 발견·판정.**

## 3. 증거 — 14개 POC (3-tier 검증)

**Tier-1 합성(논리)** · **Tier-2 실데이터+기지정답(계측기)** · Tier-3 실데이터+미지(연구, 미착수)

| POC | 검증 | 결과 |
|---|---|---|
| 1 정규화 | IPA→자질 정규화 | 깨짐 14.8%→0%. 재구표기(라링게알)는 전사 프로파일 필요 |
| 2 추출 | Wiktionary AlignedPair | 스키마 검증; descendants=계층 트리 |

| 4 시뮬→역전 | 부패 noising→denoising | **엔트로피=병합 모호성**, 자매어 3개면 완전복원 |
| 5/5b/5c 재구 | 베이지안 재구 | **얕음 ✓(PGmc 1.21), 깊음 벽; 단계적도 실패(오차누적)** |
| 6 통계 | 게이트+지평 | 게이트 sane; **탐지 지평=동원어 *개수*** (~8%서 붕괴) |
| 7 전이 | 어족간 LOFO (kaikki) | 5어족 전이 **실재하나 부분적**; 자질공간 표현 필수 |
| 9 형태론 | 닫힌부류 보존 | 얕은 데만; 깊은 절반 못 구함; 구조版은 단어트리로 측정불가 |
| 10 표현 | 이산 vs 연속 지평 | **같은 지점서 붕괴 = 한계는 정보지 표현 아님** |
| 11 타겟팅 | 안정어 부분집합 | wash(개수 한계); null 있으면 불필요; **142 우연 lookalike=체리피킹 함정** |

## 4. 깊이의 벽 — negative-results 지도 (그 자체로 기여)

심층 *증명*이 왜 막히는지를 **모호한 통념에서 측정값으로** 전환:
- **정보 한계**(POC-10): 표현(이산=연속)·블랙박스로 못 넘음. 파괴된 정보는 복원 불가.
- **동원어 개수 한계**(POC-6/11): 지평은 비율 아닌 *개수*(~15-20쌍); 안정어 타겟팅은 wash.
- **단계적 재구 실패**(POC-5c): 점추정 연쇄가 오차 누적으로 상쇄.
- **형태론 비구원**(POC-9, B-scout/Georg): 형태소는 어휘보다 *더* 우연충돌 가능; 패러다임 붕괴(Robbeets).
- **두 정보손실 기제**: 어휘교체(공통조상 단어 소멸) + 음운병합(구별 소멸).

→ 한-타밀/일본(~5천년+)은 보존율이 지평 밑 → 모델이 "판정불가+정량한계"를 반환하는 게 정직한 답.

## 5. 입증 경계 (정직)

- **입증 완료(신규)**: 중심 원리(유형화 분해 → 계보정화 + 신호복원) + 핵심 구성요소, *차용 유형의 실데이터 검증 포함*. + 깊이 한계 지도.
- **미완**: 통합 모델(여러 일탈유형 혼합 + 절약/MDL prior)을 실데이터 end-to-end로 돌려 강한 baseline 격파. + Tier-3(실제 미지 문제 적용).
- 즉 **"원리·프레이밍 입증"=✓**, **"완성 방법론 벤치마크 증명"=다음 단계**.

## 6. 자산 (재사용 가능, 검증됨)

- **데이터**: kaikki.org(wiktextract) 오프라인 덤프 (`poc/kaikki.py`, `data/kaikki/`), rate-limit 없음.
- **정규화**: `poc/normalize.py` (재구표기→panphon IPA + 운율 분리).
- **null/지평 프레임워크**: 순열검정 + 탐지 지평 (`poc/poc6_stats.py`).
- **유형화 검출**: 차용 탐지(분절 동일성 거리), 실데이터 NB 분류기 (`poc/poc12_*`, `poc13_*`).
- **전이 prior**: 자질공간 변화 분포, 다어족 (`poc/poc7_*`).

## 7. 다음 POC — 요점 (우선순위)

각 POC: **목적 / 방법 / 성공기준 / 선결조건·비용**.

### POC-15 — 카탈로그 끌림 전체 모델 (M1 완성) ⭐ 다음 1순위
- **목적**: POC-14의 단순화(phonotactic 혼합)를 *특정 출처로의 끌림* 전체 모델로 격상 → 정밀도↑ + **접촉 네트워크** 신호 복원.
- **방법**: 출처 어휘(이탈리아/시칠리아) 카탈로그 + `Adapt(w|s)` + EM. 차용을 출처-단어 단위로 귀속.
- **성공기준**: 차용 F1 > POC-14(0.76), 그리고 출처 식별(어느 단어가 어느 이탈리아어에서)이 gold와 일치.
- **선결조건**: 의미정렬된 이탈리아 어휘(kaikki Italian, 또는 Maltese 어원의 출처형) — 중간 비용.

### POC-16 — 둘째 일탈 유형으로 일반성 확인
- **목적**: "일반 기제가 차용 말고 다른 유형도 흡수하나" 검증.
- **방법(저비용)**: M2 산발변화(음위전환 등) 닫힌 라이브러리를 혼합에 추가, MDL이 진짜 산발은 채택·가짜는 기각하는지.
  (또는 M1에 유추: 패러다임 cell을 카탈로그에 추가.)
- **성공기준**: 실제 산발변화 수용 + 무작위 가짜 기각 + 규칙신호 정화.
- **선결조건**: 산발=거의 없음(최저비용). 유추=패러다임 데이터(UniMorph) 필요.

### POC-17 — Tier-3 첫 적용 = 첫 *연구 결과* ⭐⭐
- **목적**: 검증된 도구를 실제 미지 문제에. **첫 방어가능·신규 연구 결과.**
- **방법**: 한국어(±일본어)에서 **한자어(Sino-xenic) 층을 유형화로 분리** → 고유어 잔여층 신호를 **null 지평**에 대고 "판정가능 깊이 안/밖 + 정량 한계 D" 보고.
- **성공기준**: 한자어층을 gold(문서화) 대비 정확히 분리; 잔여층 신호의 p값·지평을 *수치로* 보고(긍정이든 "판정불가+한계"든).
- **선결조건**: 한국어/일본어 어휘 + 한자(Sino) 출처 데이터. (Georg의 "노이즈 구별불가" 반론을 *수치로* 다룬 최초가 될 수 있음.)

### POC-18 — 통합 end-to-end + baseline
- **목적**: "원리 입증" → "방법론 입증"(저널급). 여러 일탈유형 혼합 + 절약 prior를 실데이터서 강한 baseline 격파.
- **선결조건**: 위 POC들의 구성요소 통합.
